# Online Quiz & Assessment System

## 📌 Description
This project is a Java-based GUI application designed to conduct online quizzes and evaluate users efficiently. It provides separate modules for Admin and Student with real-time scoring and time-based quiz functionality.

---

## 🎯 Objectives
- Create an interactive quiz system for students
- Automatically evaluate answers and display results
- Provide admin control to manage questions

---

## 🚀 Features

### 👨‍💼 Admin Module
- Login using:
  - Username: admin
  - Password: 1234
- Add new questions
- Update existing questions
- Delete questions
- View all questions
- View student marks/results
- Logout and Exit options

---

### 👨‍🎓 Student Module
- Fill student details before starting quiz
- Attend multiple-choice questions
- Timer-based quiz system
- Real-time score calculation
- Automatic result display after submission

---

## 🛠 Technologies Used
- Java (Core Java & OOP Concepts)
- Swing (GUI)
- MySQL (Database)
- JDBC (Database Connectivity)
- NetBeans IDE

---

## ⚙️ How to Run

1. Import `database.sql` into MySQL
2. Open project in NetBeans
3. Update database credentials in `ConnectionProvider.java`
4. Run the project
